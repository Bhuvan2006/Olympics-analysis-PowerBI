#  Olympics Athletes Analysis ‚Äì Power BI Project

## About the Project

This Power BI report analyzes **Olympics athlete events** over the years, focusing on:

* Medal distribution by country and athlete
* Participation trends across years
* Gender ratio analysis (Male vs Female participation)
* Country-level performance comparisons
* Sports participation breakdown

The dashboard provides insights into how athlete participation and medal counts evolved over time, highlighting patterns in performance and gender representation.

##  How to Use

1. Clone or download this repository.
2. Open the `athletes_events_analysis.pbix` file in **Power BI Desktop**.
3. Explore dashboards with filters (Year, Sport, Country, Medal Type, Gender).
